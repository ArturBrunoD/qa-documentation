# Documentacao de QA - Testes Manuais e Automatizados

Este repositorio contem uma documentacao extremamente abrangente para atividades de Garantia de Qualidade (QA), incluindo:

- Fluxos de trabalho e ciclo de vida do bug
- Dezenas de User Stories (formato INVEST)
- Mapas mentais para planejamento de testes
- Centenas de casos de teste (step-by-step e BDD)
- Planos de teste e estrategias de automacao
- Relatorios de execucao e metricas
- Diretrizes e atas de reuniao
- Testes de API, performance e seguranca
- Exemplo de codigo Java com JUnit (mais de 200 testes)

## Estrutura

- `docs/workflow/` – processos e metricas
- `docs/user-stories/` – 15 historias de usuario
- `docs/mindmaps/` – 5 mapas mentais
- `docs/test-cases/step-by-step/` – 20 casos de teste detalhados
- `docs/test-cases/bdd/` – 10 features BDD (Gherkin)
- `docs/test-plans/` – planos de teste
- `docs/reports/` – relatorios de execucao
- `docs/guidelines/` – boas praticas e checklists
- `docs/meeting-minutes/` – atas de 10 reunioes
- `docs/apitests/`, `docs/performancetests/`, `docs/securitytests/` – testes especializados
- `src/test/java/com/qa/` – codigo de automacao (JUnit)

## Como usar

1. Clone o repositorio.
2. Navegue pelos arquivos para entender os processos de QA.
3. Os casos de teste podem ser importados para ferramentas como JIRA, Zephyr, etc.
