# ?? Documentaçăo de QA – Testes Manuais e Automatizados

Este repositório contém uma documentaçăo **extremamente abrangente** para atividades de Garantia de Qualidade (QA), incluindo:

- Fluxos de trabalho e ciclo de vida do bug
- Dezenas de User Stories (formato INVEST)
- Mapas mentais para planejamento de testes
- Centenas de casos de teste (step-by-step e BDD)
- Planos de teste e estratégias de automaçăo
- Relatórios de execuçăo e métricas
- Diretrizes e atas de reuniăo
- Testes de API, performance e segurança
- Exemplo de código Java com JUnit (mais de 200 testes)

## ?? Estrutura

- docs/workflow/ – processos e métricas
- docs/user-stories/ – 15 histórias de usuário
- docs/mindmaps/ – 5 mapas mentais
- docs/test-cases/step-by-step/ – 20 casos de teste detalhados
- docs/test-cases/bdd/ – 10 features BDD (Gherkin)
- docs/test-plans/ – planos de teste
- docs/reports/ – relatórios de execuçăo
- docs/guidelines/ – boas práticas e checklists
- docs/meeting-minutes/ – atas de 10 reuniőes
- docs/apitests/, docs/performancetests/, docs/securitytests/ – testes especializados
- src/test/java/com/qa/ – código de automaçăo (JUnit)

## ?? Como usar

1. Clone o repositório.
2. Navegue pelos arquivos para entender os processos de QA.
3. Os casos de teste podem ser importados para ferramentas como JIRA, Zephyr, etc.
